## Image

### Using Markdown

The [markdown syntax](https://www.markdownguide.org/basic-syntax/#images) lets you add images using the following code:

```md
![title](/path/image.jpg)
```

Image files must be less than 5 MB. For larger files, host them on a service like [Cloudinary](https://cloudinary.com/) or [S3](https://aws.amazon.com/s3/) and embed the URL.

### Using embeds

For more customizability, use [embeds](/writing-content/embed) to add images:

```html
<img height="200" src="/path/image.jpg" />
```

### iframes

An iframe loads another HTML page within the document. Use it most commonly for embedding videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
